# Hospital_Management_System_SQL_Project

SQL Project to retreive usefull insights

Managing the hosital operations includes doctor's schedule, patient' appointments and managing medical histories of the patients. This Project aims at how these day to day operations can be managed easily utilizing databases.

Database used for this project is **MySQL**.

This database contains 8 tables:

1. hospitals
2. doctors
3. patients
4. appointments
5. departments
6. medications
7. prescriptions
8. rooms
